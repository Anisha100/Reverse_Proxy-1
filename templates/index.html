Reverse Proxy project by Team Unproxy

Maintaining student attendance has been a problem in various educational institutions. Some has resorted to using manual attendance, which is very vulnerable to proxy attendances. Some are using biometric attendance systems which is not efficient and so on.
Under the theme of education, reverse-proxy is a web-based framework that can be used to handle student attendance in an institution which is more seamless than the currently used digital attendance systems. It can be used in both online education and offline education modes seamlessly.
